ReviewU Webite

Summer 2016, Single-Person Project

Desciption
A review type website that provides local locations ratings and reviews.
Details
  • - ReviewU is a rating&review type website that is aimed to provide information and reviews of locations that are located near user’s current geographical position, nearby recommendations provided at the home page.
  • - Website provides user registration. All users could search and check locations’ info/reviews. Registered users could write reviews to locations they had been to before and post unlisted locations to the website.
  • - Users could use simple or advanced search filter to find any interested locations that match given criteria.
  • - Website UI provides responsive and user friendly interface to various user devices, from mobile to desktop.
  • - Also exposes a set of Restful APIs upon internal data models/database for querying and consuming data.
Stack
  • Node.js
  • Express.js
  • MongoDB

  • Heroku
